## About

A movie and TV series discovery app using TMDB API. Built this because I wanted to practice React hooks, API integration, and responsive UI design.

## Tech Stack

React, TMDB API, CSS, Axios

## Getting Started

```bash
git clone https://github.com/shashhii/React-Entertainment-Hub.git
cd React-Entertainment-Hub
```
